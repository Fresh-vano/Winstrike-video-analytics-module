## Установка и запуск

### Frontend
Для установки всех зависимостей проекта требуется выполнить:

**npm:**

```sh
npm install
```

**yarn:**

```sh
yarn add all
```

Запуск производится командой:
```sh
npm start
```


### Backend
Для установки всех зависимостей проекта требуется выполнить:
```sh
pip install -r requirements.txt
```

Запуск производится запуском файла `backend.py`
